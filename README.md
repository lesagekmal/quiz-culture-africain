# Quiz Culturel Africain

Quiz interactif sur la culture africaine avec différentes catégories.

## Fonctionnalités
- 10 catégories de questions
- Mode entraînement
- Classement des scores
- Interface responsive
- Partage sur les réseaux sociaux

## Déploiement
L'application est disponible sur : https://lesagekmal.github.io/quiz-culture-africain/
