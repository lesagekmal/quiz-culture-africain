Politique de Confidentialité

Dernière mise à jour : 23 janvier 2025

1. Données Collectées

Nous collectons uniquement :

  • Votre pseudo (optionnel)
  • Vos scores de jeu
  • Vos préférences de catégorie

Tout est stocké localement sur VOTRE appareil.

2. Pas de Base de Données

Contrairement à la plupart des sites, nous n'avons PAS de serveur qui stocke vos données.

Vos informations restent sur votre ordinateur/téléphone dans le navigateur.

3. Classement Public

Seuls votre pseudo et votre score apparaissent dans le classement Top 10.

Ce classement est visible par tous les joueurs.

4. Comment Supprimer Vos Données

Pour supprimer toutes vos données :

  1. Ouvrez les paramètres de votre navigateur
  2. Allez dans "Données de navigation"
  3. Cliquez sur "Effacer les données"
  4. Cochez "Données des sites"
  5. Confirmez la suppression
